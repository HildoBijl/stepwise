# Expressions



## Defining the expression tree

- nodes: defines the tree.


## Input of an expression tree

- creation: helper functions to quickly set up the tree.
- parsing: turning a string into an expression.


## Manipulation of an expression tree
- manipulation: basic operations for multiple expressions, like adding, subtracting, etcetera.
- simplification: reducing a single expression to a different form.


## Inspection of an expression tree
- checks: inspecting a single expression for a property.
- comparisons: inspecting two expressions for some similarity.


## Exporting the expression tree

- printing: turning an expression into a string.
- serialization: turning an expression into a storage value.
