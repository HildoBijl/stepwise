# Number utility functions

All utility functions related to numbers are in this folder. These are grouped as follows.

## Core numbers functionalities

- [checks](./checks.ts) has checks on whether a variable is a number, and/or to ensure that it is.
- [comparisons](./comparisons.ts) has comparison functions between two different numbers.

## Applied number functionalities

- [limiting](./limiting.ts) contains general operations related to limiting numbers to a range. Think of the `mod` operator, or the `clamp` function.
- [rounding](./rounding.ts) has operators relating to bounding numbers in a certain way, with a given number of decimals or digits.
- [angles](./angles.ts) can calculate with angles: `deg2rad` and `rad2deg`.
- [random](./random.ts) can generate random numbers in various ways.
