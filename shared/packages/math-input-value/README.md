# Math input value

This package describes a data structure that supports the input of mathematics expressions and equations. It is still under development.

Important parts of the package are:
- types: the way the input value is set up.
- utils: support functions for exploring/manipulating the input value.
- parsing: all tools to turn a string into a math input value.
- normalization: turn the input value into the input state and vice versa: normalization and hydration.
- cursorMovement: updating the cursor position within the math input value.
- keyPresses: updating the value of the math input value based on key presses.
